## ◈ About

I'm a third-year EE student at **NSUT Delhi** who chose to pursue software engineering — not because it was the easy route, but because I genuinely enjoy it. I love breaking down hard problems, thinking through the logic, and writing code that works cleanly.

I'm drawn to **SDE work** — designing systems, thinking about edge cases, writing code that scales. DSA isn't just interview prep for me; I find real satisfaction in solving problems efficiently. My EE background taught me to approach things from first principles — and that mindset carries directly into how I architect software.
